Ajouter un EnrichTool¶

Pour ajouter un EnrichTool, vous pouvez implémenter les méthodes par type d’ioc disponibles dans EnrichTool.

Les méthodes doivent retourner un dictionnaire de cette forme :

{"iocType" : "{type d'ioc}", "iocValue" : "{votre ioc|Not Handleled}", report : {rapport}}

Ensuite, il faut ajouter une instance de la classe fille de EnrichTool dans EnrichApp. Pour cela il faut importer la classe de votre EnrichTool dans l’app puis l’ajouter dans le constructeur (__init__). Voir plus de détails dans la page EnrichApp.
